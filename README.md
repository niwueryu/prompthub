# PromptHub Pro

一个简洁的 AI 提示词管理工具，支持深色模式、标签分类、搜索、回收站和 GitHub 云备份。

## 🚀 网址
https://niwueryu.github.io/prompthub/

## ✨ 功能
- 📝 新建、编辑、删除提示词
- 🔍 搜索和多标签筛选
- 🌙 深色/浅色模式切换
- 🗑️ 回收站 & 恢复
- ☁️ 一键备份到 GitHub & 云端恢复
- 🔐 密码保护

## 📦 如何使用
1. 打开网址，输入密码（默认 1234）
2. 点「＋新建」添加你的提示词
3. 点「⚙️ 设置」填入 GitHub Token
4. 点「☁️ 备份」自动保存到云端
